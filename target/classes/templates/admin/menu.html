Database Schema Management


        


        
        

            

                
Table Summary

            

            

                

            

        


        
        

            Warning: Schema management operations can delete data and modify database structure. 
            Use with caution!
        


        
        

            
            

                

                    

                        
1. Drop Tables

                    

                    

                        
Drop all tables, views, and sequences. This will permanently delete all data!

                        

                            
                        

                    

                

            


            
            

                

                    

                        
2. Create Tables

                    

                    

                        
Create all tables, sequences, triggers, and views. This will recreate the database schema.

                        

                            
                        

                    

                

            


            
            

                

                    

                        
3. Populate Tables

                    

                    

                        
Insert sample data into all tables. This will add sample customers, products, orders, and payments.

                        

                            
                        

                    

                

            


            
            

                

                    

                        
4. Query Tables

                    

                    

                        
Display summary information about all tables and views, including row counts.

                        Query Tables
                    

                

            

        


        
        

            

                
Instructions

            

            

                
    
                    
  1. Drop Tables: Removes all tables, views, and sequences. Use this to start fresh.
    2. 
                    
  2. Create Tables: Creates the complete database schema (tables, sequences, triggers, views).
    3. 
                    
  3. Populate Tables: Inserts sample data into all tables. Make sure tables exist first!
    4. 
                    
  4. Query Tables: Shows a summary of row counts for each table and view.
    5. 
                

                
Typical workflow: Drop Tables → Create Tables → Populate Tables → Query Tables
